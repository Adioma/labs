Visual Storytelling

Visual storytelling is the art of imagining visual analogies between commonplace things we see every day and abstract things that are often the subject of an infographic. Can you imagine how exactly blood flows inside your body? You have never seen it, but you have seen the flow of traffic. So here's an analogy.

  

Graphs and Charts

Graphs and charts take data from abstract to concrete. The trick is knowing which chart to use. Luckily, learning that part is easy with Illustrator charting tools. While making a chart is the first step in making things visual, if you have a lot of charts they are not that interesting to look at unless you make the abstractions more concrete.

You can overlay charts with analogies, more literal analogies, and if you like communicating very literally you can even use allegories. The later are the most powerful for most subjects.

  

Designing Illustrations

Illustrations form strong memories of an image. And of course you want your readers to remember the infographic. So it's worth the time to illustrate. There are two basic ways: 1. tracing another image and 2. building an illustration out of simple shapes. For things that have a lot of detail and irregular line, tracing works better. For a more geometric look, the shapes work great.

  

Icon Design

Icon design has been around since the Egyptians. Every public restroom and road sign still makes use of the stick man.

So if icons are such a universal and fast way to communicate why can't we say everything with icons? Using the Shape tool and the Pathfinder tool in Illustrator you can create any icon.

  

How To Work With Data

Data usually come in some raw form. So the two things we can do about it are: 1. Set up a methodology for sorting it; 2. Sort it using software or custom scripting. While the latter requires some coding skills, the former is easy to do.

  

Infographic Hacks

Illustrator has so many features that the first time you open it you might think you have no chance to ever learn everything. The good news is you don't have to. The things necessary for infographic making are 1. Shape building 2. Pen tool (for tracing) 3. Pathfinder for combining shapes 4. Expand function (for making negative stroke in icons) 5. Color and gradient 6. Graph tool Yes, there are so many more cool things you can do, but with just these 6 you could be already making award-winning infographics. Start small. But really start.
